# ركيز — Rakiz Digital Agency

A modern, bilingual (Arabic/English) landing page for **Rakiz** — a digital growth agency serving SMEs in Oman and the GCC.

## Stack

- **Framework** — Next.js 14 (App Router)
- **Styling** — Tailwind CSS
- **Animations** — Framer Motion
- **Icons** — Lucide React
- **Fonts** — Inter · IBM Plex Mono · IBM Plex Sans Arabic

## Features

- Full RTL/LTR bilingual support (Arabic ⇄ English) via React Context
- Light-mode, Apple/Resend-inspired design system
- Interactive dashboard playground (Desktop/Mobile · Light/Dark preview)
- Custom syntax-highlighted integration examples
- Responsive across all breakpoints
- Zero external i18n dependencies

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Deployed on **Netlify** via the `@netlify/plugin-nextjs` adapter. See `netlify.toml` for build configuration.
